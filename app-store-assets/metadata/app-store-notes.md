1. **Screenshots**:

   - Dark mode screenshots optional but recommended

2. **App Icon**:
   - App Store icon: 1024 x 1024px (provided in app-store-assets/icons/app-icon.svg)
   - Various sizes for iOS devices will be generated by Xcode

3. **App Preview Video** (Optional):
   - 15-30 second preview video showing key features
   - Can be created later after initial release

4. **Required Information**:
   - Privacy Policy URL (host the HTML document we provided)

   - Pricing: Free

5. **App Review Information**:

## Checklist Before Submission

   - [ ] All features work as expected
   - [ ] No crashes or bugs
   - [ ] App performs well on target devices

2. **App Content**:
   - [ ] No placeholder or dummy content
   - [ ] All text is properly formatted and free of typos
   - [ ] No intellectual property violations

3. **User Experience**:
   - [ ] App follows platform design guidelines
   - [ ] Navigation is intuitive
   - [ ] Accessibility features work properly

4. **Technical Requirements**:

5. **Legal Requirements**:
