## Required Screenshot Dimensions

### Apple App Store (6.5" Display)
- iPhone 13 Pro Max, 12 Pro Max: 1284 x 2778 pixels

### Google Play Store
- Phone: 1080 x 1920 pixels
- 7-inch tablet: 1080 x 1920 pixels
- 10-inch tablet: 1080 x 1920 pixels

## Capturing Instructions

1. Run the app on a simulator/emulator or physical device with the correct screen dimensions
2. Navigate to each screen listed above
3. Take a screenshot using the device's screenshot functionality
4. For the best quality in the App Store, use devices with the largest screens
5. Ensure no personal data is visible in the screenshots
6. Make sure the UI is in English for the primary screenshots
7. Consider creating localized screenshots if you plan to publish in multiple languages
